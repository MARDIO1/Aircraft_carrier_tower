# 航空母舰塔楼 - 航模地面站控制软件

## 项目概述
这是一个用于控制航模设备的Python地面站软件，通过UART串口与航模进行通讯。

## 功能特性
- 通过UART串口与航模通讯，波特率115200
- 键盘控制：空格键切换总开关，数字1/2选择预设状态
- 实时控制台GUI显示
- 多线程架构，确保响应性能

## 硬件要求
- USB转TTL串口模块
- 航模设备支持UART通讯

## 软件要求
- Python 3.8+
- uv包管理器

## 安装依赖
```bash
uv sync
```

## 运行程序
```bash
uv run python src/main.py
```

## 操作说明
- **空格键**: 切换总开关 (0/1)
- **数字1**: 预设状态1 (开关=1, 风扇=1000, 舵机=[500,500,500,500])
- **数字2**: 预设状态2 (开关=1, 风扇=2000, 舵机=[1000,1000,1000,1000])
- **Ctrl+C**: 退出程序

## 通讯协议
地面站发送数据格式：
```
0xAA + uint8[1]总开关 + int16[1]风扇转速 + int16[4]四个舵机目标角度 + 0xBB
```

## 项目结构
```
src/
├── main.py          # 主程序入口
├── initial.py       # 初始化模块
├── protocol.py      # 通讯协议配置
├── playerInput.py   # 键盘输入捕获
├── UART_send.py     # 串口数据发送
└── terminal_GUI.py  # 控制台界面
```

## 技术架构
- 使用多线程实现并发处理
- terminal_GUI和playerInput在同一线程
- UART_send在独立线程
- 线程间通过共享变量传递数据
